## 1. 全局风格（定“基调”、切换浏览界面背景）🎨

目标：一次性把主题、布局、字体、背景、特效定好，为后续“模板个性化”和“状态项目”打底。

- 主题与配色
  - 选择主题：暗黑 🖤 / 霓虹 🌈 / 玻璃 🧊 / 手账 📒 / 像素 🕹️ / 蒸汽朋克 ⚙️
  - 更换主题会应用该主题默认圆角/阴影/配色，建议先定主题再微调
- 布局与比例
  - 标签:数值 比例（如 4:6 / 3:7），让“值”更显眼
  - 布局模式：标签在左 / 堆叠 / 居中 / 双列 / 反转
- 字体与排版
  - 标签/数值：字体、字号、字重、斜体、大写、字距、倒影
  - 导入自定义字体：粘贴 Google Fonts 或第三方 CSS 链接，稍等即生效
- 背景与特效
  - 背景层：纯色/渐变/图片/URL，可叠加多层（层次更丰富）
  - 特效：星星 ✨ / 碎屑 🔶 / 花瓣 🌸（密度/频率/颜色/发光/自定义图片）
- 设备预览
  - 点击“📱 手机端”切到 375px 预览；再次点击回“💻 电脑端”
  - 仅影响预览，不改变导出代码

✅ 建议顺序：先主题 → 再布局/比例 → 字体 → 背景/特效 → 用手机/电脑预览检查可读性

## 2. 模板个性化（定“样式语言”）🧩

目标：设置标题/头部、默认项目风格、区域行为与动画，确保“好看 + 好用”。

- 标题 / 头部
  - 开关“显示标题”，输入标题（例：角色状态）
  - 标题样式：字号、字重、对齐、字距、下划线（样式/颜色/厚度/偏移）
  - 颜色模式：主题色 / 纯色 / 渐变
  - 标题图标：内置图标或自定义 URL（可加圆角/发光/阴影/羽化）
  - 头部背景：纯色/渐变/玻璃；可设最小高度、上下间距、标题间距
- 动画与氛围
  - 进入动画 / 循环动画（轻度使用更耐看）
  - 光晕 / 投影（别太猛，保证可读性）
- 默认项目风格（全局默认，可被单项覆盖）
  - 进度条：风格（标准/荧光/斜纹/玻璃）、动画（增长）
  - 百分比显示位置：居中/徽章/气泡/尾随/吐司/左/右
  - 分割线：实线/虚线/渐变
  - 项目卡片：默认卡片背景（纯色/渐变/图片/URL）与阴影强度
  - 值框与偏移：默认值框宽度、值框偏移、项目整体偏移
- 区域（分组 + 折叠）
  - 区域标题样式（纯色/渐变）、图标（内置/URL）
  - 点击目标：整个头部 / 仅标题文字 / 仅按钮（交互最顺手）
  - 导出默认折叠 ✅（信息量大时更清爽）
  - 展开动画：翻转/擦除/弹出/下划线/涟漪/故障风（速度/强度可调）
  - 区域外观：边框/圆角/内边距/阴影；区域体背景（无/纯色/渐变/图片/URL）

✅ 建议：标题尽量简洁；动画做“轻”，保证内容优先和可读性

### 强烈推荐：多图层自定义背景（零基础也能叠出质感）🎯
- 开启位置
  - 2 模板个性化 → 背景模式 选择“多图层自定义背景”。随后会出现“图层列表”和“组件列表”编辑区（可折叠）。
  - 手机/电脑预览按钮“📱 手机端”仅影响右侧预览，不改变导出结果。

- 图层类型（可无限叠加，支持上下顺序）
  - 颜色图层：设置纯色与不透明度（0~1）。建议作为“打底层”，提升统一性和对比度。
  - 渐变图层：风格 线性/径向/锥形；线性用“角度(°)”控制方向；径向/锥形用“方向/位置”。可分别设置“起色/终色”。适合做光感、聚焦、氛围过渡。
  - 图片图层：粘贴图片 URL（需要先把图片转换成网址格式，默认 cover/center/no-repeat）。比如叠纸张/噪点/纹理/光斑等素材，透明度导入后依然可调节。
  - 顺序与透明度：每层都可“上移/下移”，并用“透明度”微调融合度。上方的层会覆盖下方层。

- 背景组件（贴纸/装饰，可拖拽摆放）
  - 添加组件：点击“添加组件 URL”，粘贴图标/贴纸/徽章图片链接。
  - 基本参数：宽度(%)、透明度、显示/锁定 开关。锁定后无法被拖动，避免误触。
  - 拖拽模式：打开“拖动模式”开关后，可直接在右侧预览“抓取并移动”组件，以百分比坐标定位。界面底部会提示“拖动模式：开启”。
  - 应用建议：用半透明 PNG 叠光晕/徽章/贴纸，细节提升明显但不会喧宾夺主。

- 进阶：裁剪与边缘特效（每个组件专属）
  - 裁剪：上/右/下/左（%）四向内切，快速适配素材边缘或遮掉无用区域。
  - 特效（可多选叠加）：
    - 圆角：设置半径 px，配合 PNG 边缘可更柔和。
    - 发光：强度 0~50，基于主/辅色做 drop-shadow 高光，营造霓虹/魔法感。
    - 阴影：一键高质感投影，增强层次。
    - 柔化边缘：羽化强度 0~50，用遮罩柔化边界，贴纸更融入背景。
  - 提示：参数在“组件卡片”中可见；所有修改都会实时反映到预览和导出代码。
  - 机制参考：[Ny.Render.buildBgComponentsHTML()](js/app.render.js:4438) · [setupBgComponentDrag()](js/app.render.js:4454)

- 导出与酒馆（保证所见即所得）
  - “状态栏 HTML（实时生成）”会包含你的所有图层与组件结构；“下载酒馆正则（JSON）”同样适配该结构并在酒馆内替换成同样的 DOM。
  - 网络图片：导出后仍按 URL 加载，确保链接稳定可访问；如需离线，请改用可长期可用的图床。
  - 引擎说明：背景包装层计算由 [Ny.State.computeBackground()](js/app.state.js:336) 负责，当选择“多图层”时内联背景会置为透明，由渲染函数插入图层与组件 DOM：[Ny.Render.buildBgLayersHTML()](js/app.render.js:4396) · [Ny.Render.buildBgComponentsHTML()](js/app.render.js:4438)。

- 新手三套高质感配方（按顺序叠加）
  1) 电影氛围
     - 颜色图层：#0B0F1A，不透明度 0.85
     - 径向渐变：位置 center，起色 #0B0F1A → 终色 透明，角度任意
     - 图片图层：细腻噪点/胶片颗粒 PNG，透明度 0.12
     - 组件贴纸：小光斑 PNG，宽度 12%，发光 30
  2) 霓虹玻璃
     - 线性渐变：角度 135°，起色 #00F7FF → 终色 #FF00A6，透明度 0.25
     - 图片图层：玻璃磨砂纹理 PNG，透明度 0.22
     - 组件贴纸：电路/芯片图标，宽度 10%，发光 40，阴影 开
  3) 纸感手账
     - 图片图层：米黄色纸张纹理 JPG，透明度 0.35
     - 线性渐变：角度 180°，起色 #00000080 → 终色 透明，做顶部压暗
     - 组件贴纸：手绘风格元素（纸胶带/便签），圆角 6px，阴影 开

- 实用建议
  - 读屏可读性：始终用颜色/渐变“先打底”，并通过透明度控制，确保标签/值文字对比度足够。
  - 性能：2~4 个图层 + 少量组件已足够精致；图片请尽量压缩到千级像素、100~300KB 量级。
  - 调整顺序：先做图层（大背景），再摆放组件（小装饰），最后微调透明度与动画。
  - 常用图床推荐：https://catbox.moe/

## 3. 状态项目（添加与定制“状态栏的项目”）📋

目标：把要展示的信息录入“项目”，支持文本/长文/进度条/分割线/区域。

- 项目类型（点击“新增”）
  - 文本 📄：标签 + 值（例：地点 = NY预设帖子内）
  - 长文字 📝：多段落描述；可开启打字机/淡入/上滑等动画
  - 进度条 📊：拖动百分比；风格/动画/百分比位置按需选择
  - 分割线 ➖：用于视觉分组
  - 区域 🗂：可折叠的分组容器，内部可继续添加子项目
- 编辑与排序
  - 每条项目右上角：⬆ / ⬇ / ✖（上移 / 下移 / 删除）
- 单项外观覆盖（更灵活）
  - 卡片背景（纯色/渐变/图片/URL）、阴影强度
  - 值框宽度/偏移与项目整体偏移
  - 长文字的行距、首行缩进、内边距、动效时序
- 进度条交互
  - 在预览里“点击进度条”可切换百分比显示/隐藏（导出后仍然有效）
- 区域组织力
  - 区域内可添加文本/进度条/长文/分割线等子项目
  - 合理设置“点击目标”与“默认折叠”，信息密度高也能优雅呈现

✅ 建议：信息多时先用“区域”分组，再细化每项样式；分割线仅作视觉辅助

## 💡 常见问题与小贴士

- 主题切换会应用新默认外观（颜色/圆角/阴影）：先定主题再微调
- 自定义字体加载可能稍慢：预览等 1~2 秒；导出已内置必要加载策略
- 区域点击没反应：检查“点击目标”是否选成“仅按钮/仅标题文字”，对准点击即可
- 进度条百分比开关：预览/导出中点击进度条即可显隐
